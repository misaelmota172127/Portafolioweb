Conocimiento general

Estos son algunos de los lenguajes y frameworks de desarrollo web que he utilizado.

HTML

El lenguaje básico de marcado utilizado para estructurar el contenido en la web.

CSS

Se utiliza para dar estilo y presentación a las páginas web.

PHP

Es un lenguaje de scripting del lado del servidor usado para crear sitios web dinámicos.

C#

C# es un lenguaje de programación de Microsoft usado para crear aplicaciones web, de escritorio y móviles.

C++

C++ es un lenguaje de programación de propósito general conocido por su rendimiento y control sobre los recursos del sistema.

JavaScript

Lenguaje de programación que permite añadir interactividad y dinamismo a las páginas web.

Bootstrap

Framework popular para crear sitios web responsivos y con buen diseño.

.NET

Framework desarrollado por Microsoft para construir aplicaciones robustas y escalables.

React

Biblioteca de JavaScript para construir interfaces de usuario interactivas.
